CSS

CSS is precisely created so it can provide extraordinary user experience. The product pictures are aligned and listed one after another. When user puts a computer mouse on the picture it is hovered and the product name and price is shown. There are New arrivals and also products that are on sale. For the products on sale, discounted prices are visible.

The following code is from the css project file, it shows how the previously described effects are achieved.

.products { margin-bottom: 60px; } .products h1 { text-decoration: none; padding-bottom: 25px; } .productimages { display: inline-block; margin-right: 17px; background-color: #ddd; } .productrows { margin-bottom: 10px; } .cleaner { clear:both; } .strikethrough { text-decoration: line-through; font-weight: 100; } .discountPrice { font-size: 22px; font-weight: 500; }

JQueri

Crucial part of the website technologies is jQuery. JQuery Ajax is available on the main page of the store and email validation is used for the contact form. JQuery Ajax enables the function for showing more text, it a user wants to see. On the other hand for email validation on the contact form is used pattern (that should be fulfilled). The pattern is "[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,3}$". This means that the email should have text@text.text
